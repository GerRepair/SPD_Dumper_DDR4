Third-Party License Notice: InpOutx64.dll / InpOut32
====================================================

Component:
  InpOutx64.dll / InpOut32

Purpose:
  Windows DLL and driver for direct hardware I/O port access from user-mode
  applications. This project is used by this application only as a third-party
  hardware access component.

Upstream / Attribution:
  InpOut32 was originally developed by Logix4U.
  The x64 port and maintained public download were provided by Phillip Gibbons / Highresolution Enterprises.

Upstream information:
  Highrez / InpOut32 and InpOutx64 download page:
  https://www.highrez.co.uk/downloads/inpout32/

License:

Notes:
  The upstream Highrez page states that version 1.5.0.1 updated the license
  terms to the standard MIT License and that License.txt is included in the
  download(s). The same page describes the project as legacy and no longer
  actively supported.

Security / operational notice:
  InpOutx64.dll loads/uses a kernel driver to provide low-level hardware access.
  This application requires administrator privileges for that access. Use only
  trusted copies of InpOutx64.dll obtained from the official upstream source or
  from a known trusted package.
